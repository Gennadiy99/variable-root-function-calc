Создание переменной в CSS

Создание переменной в CSS выполняется при помощи селектора ":root" где указываются переменные видимые в любом месте документа. 

            :root {
              --color1: #05b1b1;
              --color2: #52339a;
              --bgc_block: #aff08e;
            }

Переменные для цветов создаются, что бы ла возможность изменить Пименяемый в элементах цвет при помощи изменения его в однойм месте, в переменной. И цвет изменится во всем докумнете. Не мужно выискивать каждый элемент и в ручную исправлять его.
Название переменным можно давай любые, (например: описывающие их назначение)

Чтобы вызвать значение переменной нужно указать конструкцию: "var(--color1)" 

            .wrapper {
              padding: 0 100px;
              background-color: var(--bgc_block);
            }

При вызывании переменной можно указывать через запятую втрой цвет, который будет появляться в случаии если произойдет какая то ошибка с переменной 

            .wrapper {
              padding: 0 100px;
              background-color: var(--bgc_block, yellow);
            }

Так же можно создать пеерменную с значением размера в пикселях например, и прменить ее к блоку: 

            :root {
              --size: 600px;
            }
            
            .block{
              width: var(--size);
            }

Свойство "calc" в "CSS"

Данной свойство помогает вычеслять значения при помощи математических операторов (+ - / *)

Суть использования свойства "calc" это функция которая помогает расчитывать Ширину или Высоту Дочернего элемента относительно Родительского элемента.

На примере ниже зададим размер дочернему элементу при помощи свойства "calc"

  • Задаем свойства Pодительскому элементу: 
                  .box-paret {
                background-color: var(--color1);
                border: 2px solid #fa3904;
              }
  • Задаем свойства Дочернему элементу: c использованием свойства "calc" 
                  .box-parent__child {
                background-color: var(--color2);
                border: 2px solid #419c2b;
                margin: 0 auto;
                width: calc(100% / 3);
              }

В свойство "calc" можно вставять CSS переменные. (которые содержат числовые значения)

Пример создание числовой переменной и применение ее в свойстве "calc"s 

          :root {
            --colNums: 4;
          }
          .box-parent__child {
            width: calc(100% / var(--colNums));
          }
