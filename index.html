Создание переменной в CSS

Создание переменной в CSS выполняется при помощи селектора ":root" где указываются переменные видимые в любом месте документа. 

              :root {
                --color1: #05b1b1;
                --color2: #52339a;
                --bgc_block: #aff08e;
              }

Переменные для цветов создаются, что бы ла возможность изменить Пименяемый в элементах цвет при помощи изменения его в однойм месте, в переменной. И цвет изменится во всем докумнете. Не мужно выискивать каждый элемент и в ручную исправлять его.
Название переменным можно давай любые, (например: описывающие их назначение)

Чтобы вызвать значение переменной нужно указать конструкцию: "var(--color1)" 

              .wrapper {
                padding: 0 100px;
                background-color: var(--bgc_block);
              }

При вызывании переменной можно указывать через запятую втрой цвет, который будет появляться в случаии если произойдет какая то ошибка с переменной 

              .wrapper {
                padding: 0 100px;
                background-color: var(--bgc_block, yellow);
              }

Так же можно создать пеерменную с значением размера в пикселях например, и прменить ее к блоку: 

              :root {
                --size: 600px;
              }
          
              .block{
                width: var(--size);
              }

Свойство "calc" в "CSS"

Данной свойство помогает вычеслять значения при помощи математических операторов (+ - / *)

Суть использования свойства "calc" это функция которая помогает расчитывать Ширину или Высоту Дочернего элемента относительно Родительского элемента.

На примере ниже зададим размер дочернему элементу при помощи свойства "calc"

  • Задаем свойства Pодительскому элементу: 
                    .box-paret {
                  background-color: var(--color1);
                  border: 2px solid #fa3904;
                }
  • Задаем свойства Дочернему элементу: c использованием свойства "calc" 
                    .box-parent__child {
                  background-color: var(--color2);
                  border: 2px solid #419c2b;
                  margin: 0 auto;
                  width: calc(100% / 3);
                }

В свойство "calc" можно вставять CSS переменные. (которые содержат числовые значения)

Пример создание числовой переменной и применение ее в свойстве "calc"s 

            :root {
              --colNums: 4;
            }
            .box-parent__child {
              width: calc(100% / var(--colNums));
            }

Работа с Припроцессором "sass" и "scss"

Для подключения "sass" нужно поработать с командной строкой и прописав путь к файлу "main.sass" указать команду "sass main.sass style.css"подключить компиляцию из файла "main.sass" в "style.css"

  • Windows + r - Вызываем командную строку Windows.
  • cmd - Указываем команду для открытия строки.
  • dir - Команда для просмотра содержимого папки.
  • cd "Название папки" - Переход в указанную папку.
  • sass main.sass style.css - Команда указывается в в конце пути к Папке где лежит файл "main.sass"
  • В резульате команды создается файл "css" куда скомпилируется код из файлала "main.sass"

Установить программу "Koala" получилось, прописать путь тоже, изменил все папки на Латиницу, заменил пробелы на нижне подчеркивание, но компиляция и обновление не работают. Выскикивает непонятная ощибка с иероглифами.

Существует 2 синтаксиса написания "sass" - "scss". Рассмотрим в чем отличие между этими синтаксисами написания кода на примерах.

  • "sass" - без кавычек круглых и фигурных, без знака ";" все разделяется отступами и табуляцией
  • "scss" - есть кавычки и ";" принадлежность тегов указывается вложенностью.

Можно создавать переменные, в которые влаживать значение цветов, онов, размеров и после применять их в стилизации Селекторов. Синтаксис при создании переменных такой:

$color-grey:rgb(113, 111, 111);

Селекторы создаются путем вложенности тегов друг в друга. Пример кода на "scss": 

#header {
            background: #ffffff;
            width: $pad;
            color: $color-grey;
            a {
              text-decoration: none;
              &:hover {
                text-decoration: underline;
                color: #df09ad;
              }
            }
          }

Это тоже самое что на "CSS" написать: 

            
              #header {
                background: #ffffff;
                width: $pad;
                color: $color-grey;
              }
              #header a {
                text-decoration: none;
              }
              #header a:hover {
                text-decoration: underline;
                color: #df09ad;
              }

Что бы использовать псевдокласс ставится знак "амперсант - &" Прмер: 

            
              a {
                text-decoration: none;
                &:hover {
                  text-decoration: underline;
                  color: #df09ad;
                }
              }

Так же мы можем создавать файл МОДУЛЕЙ,
синтаксис написания с нижнего почеркивания"_module.scss" внем прописывать "свойства": 

Код в файле модуля:
              
                html, body{
                  margin: 10px;
                  padding: 15px;
                }

А потом импортировать их в файл "main.scss" при помощи свойства: "@import 'module';"
В файле 'main.scss' уже пишем без нижнего подчеркивания. 

            Код в файле куда импортируются данные "main.scss" 
            
              @import 'module';

При компиляции файла из "main.scss" в "main.css" данные из модуля в него автоматически переносятся.

Применение "Миксин" - (функция с переменной)

Создается "Миксин" в файле "main.scss" и вызывается в любом месте файла по необходимости. После компиляции попадает в файл "main.css" во вложенном теге. 

Создание миксина:
            
              @mixin border($radius) {
                -webkit-border-radius: $radius;
                -moz-border-radius: $radius;
                -ms-border-radius: $radius;
                border-radius: $radius;
              }

Данный "Миксин" добавляет Префиксы для border

  • -webkit-border-radius: $radius;
  • -moz-border-radius: $radius;
  • -ms-border-radius: $radius;

А в параметре "Миксина" @mixin border($radius) указывается переменная "$radius"значение которой мы укажем.

Вызывается "Миксин" припомощи свойства: "@include border(10px)" Вызывается "Миксин" в теге к которому мы хотим его применить: 

            
              p.container{
                @include border(10px);
                padding: 15px;
              }

Расширение или наследование

Создается блок кода при помощи знака процент % 

            
              %myCode{
                font-size: 20px;
                border: 5cap solid #000;
                padding: 10px;
              }

Вызывается в нутри тега при помощи, "@extend %myCode;"

            
              a {
                @extend %myCode;
                text-decoration: none;
                &:hover {
                  text-decoration: underline;
                  color: #df09ad;
                }
              }

Если какой то код встречается часто то его можно внести в Блок расширения и подключать везде где необходимо.

Применение математических операций.

В sass это делается без функции "calk" (как в "CSS") а просто в нужном месте через арефметический оператор пишится действие. 

            
              %myCode{
                font-size: 20px+2px;
                border: 5cap solid #000;
                padding: 10px;
              }
